# AI Food Label Scanner

Приложение със Streamlit и EasyOCR за разпознаване на вредни съставки в храните.

## Функции

- Качване на снимка
- Снимка от камера
- OCR разпознаване на текст
- Откриване на вредни съставки
- Поддръжка на български и английски език
